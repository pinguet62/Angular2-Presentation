Encore un nouveau langage à apprendre ?

Pourquoi il faut choisir TypeScript

@CanSpeak('Hello world!')
class Person {
    private name: string;

    constructor(name: string) {
        this.name = name;
    }

    introduceYourself(): void {
        console.log('My name is ' + this.name);
    }
}

let p: Person = new Person('Kevin');
p.introduceYourself(); // My name is Kevin
(<any> p).speak(); // Hello world!

Essayons ECMAScript 6 !

class Person {
    constructor(name) {
        this.name = name;
    }

    introduceYourself() {
        console.log('My name is ' + this.name);
    }
}

let p = new Person('Kevin');
p.introduceYourself(); // My name is Kevin
Heu, c'est pareil non ?

Quant à ECMAScript 5...

var Person = (function () {
	function Person(name) {
		this.name = name;
	}
	
	Person.prototype.introduceYourself = function () {
		console.log('My name is ' + this.name);
	};
	
	return Person;
}());

var p = new Person('Kevin');
p.introduceYourself(); // My name is Kevin
Qui a compris ?

Typage facultatif

let p1: any = new Person('Kevin');
let p2 = new Person('Kevin');

Les services

  • Fonctionnalités
  • Code métier
  • Appel de webservice

AngularJS...

  • Service : propriétés = fonctions
  • Factory : retourner un objet où les propriétés sont des fonctions
  • Provider : propriété $get de type fonction retournant l'objet
angular
	.module('foo')
	.service('myService', ['otherService', function(otherService) {
		this.doSomething = function (arg) {
			// ...
		}
	}]);

angular
	.module('foo')
	.factory('myService', ['otherService', function(otherService) {
		return {
			doSomething: function (arg) {
				// ...
			}
		};
	}]);

angular
	.module('foo')
	.provider('myService', ['otherService', function(otherService) {
		this.$get = function () {
			return ...
		}
	}]);

Les pipes

Angular !

Interface

interface PipeTransform {
	transform(value: any, ...args: any[]): any;
}

Implémentation

import { Pipe, PipeTransform } from '@angular/core';

@Pipe({ name: 'upper' })
class UpperPipe implements PipeTransform {
	transform(value: string): string {
		return value.toUpperCase();
	}
}
